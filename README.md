求解器更新1.0.2，实现quickxplain冲突解释器，目前可支持的约束:
1. 基本的数学表达式 + - * / == and(&) or(|) not(~) ( )
2. 对于蕴含式，应将其转化为逻辑表达式 p -> q转化为not p or q
3. 等价式，转化为两个蕴含式
4. 复杂约束，比如某个表达式中含有两个相同变量，比如a*a > 4（尽管该约束几乎不会出现），在下一个版本实现
